Website Enhancements

Mobile Compatibility

This website is designed to be fully responsive and mobile-friendly. This means it will look good and be easy to use on any device, from desktop computers to smartphones. The layout, images, and text all adjust automatically to fit the screen size.

Try resizing your browser window or viewing this site on your phone to see how it works.

The use of the viewport meta tag and css media queries to adjust the width of certain aspects of the website to relative values were used commonly to help make this possible as well as added in visibility and reflow elements to help make it a seamless transition!

More obviously seen on the apply.html page and the about.html page.

Animation on About Page

The About page features a unique animation that enhances the user experience. When you click on a team member's photo, their bio appears with a smooth, fading transition.

Visit the About page to see this animation in action.
